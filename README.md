# 🔐 Password Generator

A simple and clean **Password Generator** built with **React** and **Tailwind CSS**.

## 🚀 Live Demo
👉 [https://dpnxhuu.github.io/password-generator-react/](https://dpnxhuu.github.io/password-generator-react/)

## ✨ Features
- Generate strong random passwords
- Built with React + Tailwind CSS
- Fast and lightweight

## 🛠️ Tech Stack
- React
- Tailwind CSS
- Vite

## 📦 Run Locally

```bash
git clone https://github.com/Dpnxhuu/password-generator-react.git
cd password-generator-react
npm install
npm run dev
```

## 📤 Deploy
```bash
npm run deploy
```
